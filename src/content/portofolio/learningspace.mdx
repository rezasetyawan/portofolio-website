## Description

Website based learning management system that have dynamic learning class and module. User able to expand their knowledge by joining available class that have learning, quiz, and submission module. User able to interact with other user in module discussion page, where user able to create and answer discussion.

## Features

### User (Student)
#### 1. Login and Register
User able to do register and login in auth page
#### 2. Joining class
User able to explore class on academies page and join desired class
#### 3. Learn class module and update progress
After join a class user able to learn every module inside it. There 3 variant of module that is learning module, quiz module, and subsmission module. User able to take quiz in quiz module. In submission module user must work with a task that given by admin. Once user finished the module, it automatically update their progress in that class. User also able to see their quiz and submission histories.
#### 4. Create and answer discussion on module discussion page
Inside module page, user able to navigate to module discussion page. In that page user able to explore other's discussion and answer to that discussion. In the other hand user able to create their own discussion.
#### 5. Manage profile
User able to change their username, fullname, and profile picture.

<hr></hr>
### Admin
#### 1. Login and Register
Admin able to do register and login in auth page
#### 2. Manage class
Admin able to see, create, update and delete a class. Admin also able do the same thing to module inside that class.
#### 3. Contribute to module discussion
Admin also able to contribute to user module discussion
#### 4. Review Student's submitted submission
Admin able to review student's submitted subsmission. Admin can either accept or reject user submission. Admin also able to leave a note and score to reviewed submission.
#### 5. Process user join class request
Admin able to either accept or reject user join class request.
#### 6. Dashboard
In dashboard page admin able to see active user stats, popular class, class count, and unreviewd submission count.

<hr></hr>
### Superadmin
#### 1. All admin features
Superadmin adble to do all what admin can do.
#### 2. Manage user
Superadmin able to managem user data, but of course there's a limitation. Superadmin only able to create new user and update user role.
